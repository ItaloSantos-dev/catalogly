# Catalogy

Catalogy é um sistema de criação de catálogo para pequenos e médios empreendedores.
A aplicação é voltada para a criação de uma vitrine de produtos, pagamento no próprio site e um dashboard geral de vendas.

## Visão do produto

- Criar uma vitrine online de produtos fácil de gerenciar.
- Permitir que o empreendedor publique produtos com imagens, descrição e preço.
- Oferecer pagamento diretamente no site.
- Disponibilizar um painel de controle (dashboard) para acompanhar vendas, pedidos e métricas.
- Leitura de Nota fiscal para atualização automática de estoque 

## Funcionalidades planejadas

- Cadastro e gerenciamento de produtos.
- Vitrine pública de produtos com busca e filtros.
- Carrinho de compras e checkout.
- Integração com provedores de pagamento.
- Dashboard administrativo com relatório de vendas e indicadores.
- Autenticação básica para lojistas.

## Tecnologias

- Java 21
- Spring Boot 4
- Spring MVC
- Spring Data JPA
- PostgreSQL (runtime)
- Lombok
